# Basic Math Buddy – C Calculator

## 📌 Project Overview

**Basic Math Buddy** is a simple calculator application developed using the **C Programming Language**. The program performs basic arithmetic operations such as addition, subtraction, multiplication, division, and modulus. It provides a menu-driven interface where users can enter two numbers, select an operation, and instantly receive the result.

This project is designed for beginners to understand the fundamentals of C programming, including variables, operators, conditional statements, functions, and user input/output handling.

## 🚀 How to Run

### Step 1: Save the Source Code

Save the program as:

```bash
calculator.c
```

### Step 2: Compile the Program

Using GCC:

```bash
gcc calculator.c -o calculator
```

### Step 3: Run the Program

Windows:

```bash
calculator.exe
```

Linux/macOS:

```bash
./calculator
```
