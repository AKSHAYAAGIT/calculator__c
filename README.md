## 📖 Project Description

Basic Math Buddy Calculator is a menu-driven calculator developed entirely using the C programming language. The project performs both basic and scientific mathematical operations through an interactive console-based user interface.

This calculator demonstrates the use of functions, loops, switch-case statements, file handling, mathematical libraries, and modular programming concepts in C. It provides a user-friendly menu that allows users to perform various calculations quickly and efficiently.

The project is designed as a mini-project for first-year engineering students to understand practical applications of C programming concepts.
