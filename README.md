# Basic Math Buddy

## Project Description

Basic Math Buddy is a simple calculator project developed using C Programming for performing arithmetic operations such as Addition, Subtraction, Multiplication, and Division.

The project demonstrates the implementation of functions, header files, conditional statements, switch-case structures, user input handling, and modular programming concepts in C.

To improve user experience, an attractive graphical interface is created using HTML, CSS, and JavaScript. The calculator provides a clean design with interactive controls for performing calculations easily.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- User-friendly interface
- Error handling for division by zero
- Modular C programming structure

## Technologies Used

- C Programming
- HTML
- CSS
- JavaScript

## Learning Outcomes

- Function implementation
- Header file usage
- Modular programming
- Switch-case statements
- Input and output operations
- Basic front-end development
